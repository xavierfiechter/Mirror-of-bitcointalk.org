__Specification Coin
- Launched 24th of April 2017
- Odarhom Core
- Max supply of 21 million coins
- 2.5 min average blocktime
- DefaultPort = 8555 - RPC
Port 40332
- Current blockchain size
approximately 1.2 GB
- Segwit and Bloom
- 10MB Blocksize
(20MB Segwit)
- MEGA-BTX (GPU)
mining algorithm
and Masternodes
- SegWit and Bloom online
- Diff Retarget: Bitcore 64_15
- Fair distribution:
airdrops (ended) + BTC claiming (ongoing)
_Segwit with
Core 0.90.9.X
The SEGWIT opens up new possibilities like the
Lightning Network,Tumblebit, Schnorr Signatures,
Confidential Transactions, Cross-chain atomic
swaps, and so on.
We are already exploring all these new functions,
and testing the water.
We expect the first implementations as fast as
possible.

PoW Algo
BitCore BTX has developed a new
algorithm named Mega-BTX designed to make
FPGA mining too expensive and make mining
efficient with GPU and CPU equipment. This
new algorithm Mega-BTX version 1.9.9.5 will be
launched September 1st, 2020 12:00 pm (UTC),
and as it was mentioned before is made mainly
to support GPU mining.
_MasterNodes
Odarhom brings along a masternode
system for BitCore. The collateral
for one masternode is 2,100 BTX.
This allows up to 10,000 masternodes
to support the network. The
masternodes receive half of all
generated bitcores. It is possible
to setup a masternode with the
minimum version 0.90.x.x or higher.
A government system is included in
the new core on and can be activated
later, if necessary.
