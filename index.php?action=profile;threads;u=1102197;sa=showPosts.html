17  Local / Bahasa Indonesia (Indonesian) / [TIPS] Pilihan Untuk Mengatur Urutan Thread di Bitcointalk on: June 27, 2018, 11:52:27 PM
Sesuai judul, saya mau berbagi trik bagaimana mengurutkan thread di bitcointalk, sesuai keinginan kita.

Di default Bitcointalk, thread yang ditampilkan di halaman paling awal adalah thread yang terakhir mendapat reply. Namun ada beberapa pilihan lain bagaimana urutan thread ditampilkan. Berikut beberapa pilihan yang ada, dan bagaimana mengaturnya.


Pertama pilih salah satu sub-forum, contoh Bahasa Indonesia:
https://bitcointalk.org/index.php?board=191.0

Kemudian tambahan pada url nya sebagai berikut, :

1. Tampilkan thread terbaru di halaman awal :
Code:
;sort=first_post;desc
https://bitcointalk.org/index.php?board=191.0;sort=first_post;desc

2. Tampilkan thread tertua di halaman awal:
Code:
;sort=first_post;asc
https://bitcointalk.org/index.php?board=191.0;first_post;asc

3. Tampilkan Thread dengan reply terbanyak di halaman awal.
Code:
;sort=replies;desc
https://bitcointalk.org/index.php?board=191.0;sort=replies;desc

4. Tampilkan Thread dengan reply paling sedikit di halaman awal.
Code:
;sort=replies;asc
https://bitcointalk.org/index.php?board=191.0;sort=replies;asc

5. Tampilkan thread dengan view terbanyak di halaman awal:
Code:
;sort=views;desc
https://bitcointalk.org/index.php?board=191.0;sort=views;desc

6. Tampilkan thread dengan view terbanyak di halaman awal:
Code:
;sort=views;asc
https://bitcointalk.org/index.php?board=191.0;sort=views;asc

7. Urutkan sesuai dengan abjad pertama username pembuat thread/TS/OP.
Quote
Jadi thread yang dibuat oleh username dengan abjad pertama berupa angka/simbol akan ada di halaman awal, dan yang dibuat oleh username dengan huruf Z sebagai abjad pertamanya akan ada dihalam paling terakhir.
Code:
;sort=starter
https://bitcointalk.org/index.php?board=191.0;sort=starter

8. Urutkan sesuai dengan abjad pertama subjek thread
Code:
;sort=subject
https://bitcointalk.org/index.php?board=191.0;sort=subject


Sekian tips dari saya semoga dapat membantu agan-agan disini agar bisa lebih berkontribusi lagi untuk forum ini.
Terima kasih
