Mineitor is a remote monitoring and management solution for ASIC miners.
With Support for large-scale farms. and multi-locations.
Supported Brands
   Avalon, Baikal, Bitfily, Bitmain, Braiins, Vnish, Cheetah Miner, Dragon Mint, Ebang, Goldshell, InnoSilicon, Lovecore, PandaMiner, StrongU, Todek, Whatsminer ,Hyperbit, iBeLink

Mineitor Features:
- Remote Monitoring
  Mineitor provides real-time data, history, and graphs of devices hash rate, temperature, fan speed, pool status, and all the data provided by the miner in one place
- Remote Control
  Mineitor can apply different configurations and settings to the miners from everywhere and anytime. It also saves the history of the configurations and settings applied by different users having access.
- Remote-browsing
  Access device's admin web panel from anywhere with Mineitor. it creates a secure tunnel between your browser and device. every action of the user will be logged.
- Customizable Alerts & Notifications:
 Create your custom-made events based on device current information and brand, model, or groups.
- Bulk operations:
  Mineitor provides the ability to apply configurations and settings to miners in batches. Miners are manageable in different groups to make this process even easier.
- Pool Lock
  Mineitor can Lock and unlock device settings. with lock enabled on the farm, it is impossible to change the device configuration from the device panel or any other methods.
  Even if the device is infected with viruses it prevents configuration changes. (SSH access needed)
  without ssh access Mineitor checks device configuration within 10-second intervals and if the device's pool setting changed it alerts and reverts the setting.
- Automated schedules and actions
 Define your actions and jobs with customizable timing to be executed based on the device's information. e.g. reboot or stop CGminer if the temperature rises higher than 80 degrees or change pool at midnight.
- Grouping
  Create custom groups to monitor the overall hash rate, alerts, maximum and minimum temperature of devices of that group.
-widgets and Monitoring-wall
 Create a customizable dashboard and monitoring wall with editable widgets to see what's happening in your farm/farms in real-time.
- Sections
 It is possible to define the locations and shelves and coordinates of devices inside your farm. all data is accessible based on the locations you define. with overall information.
- Telegram bot
 get alert and notifications of events and the overall status of your farms in telegram
- User management
 Give access to other teammates for all farms or only specific devices.
- Heat Map
  Locate hot spots within shelves and faulty devices in one view. locate the problems faster!
- Logging System
  Any action done by the user form within Mineitor will be logged and can be reviewed later.
- Internal user management
  Manage Access with rules. define access to a specific menu or feature for a group of devices or a farm.
- Maintenance and Repair
  Mineitor maintenance enables you to track requests and schedule preventive maintenance easily with a user-friendly interface. technicians and responsible can be assigned to sections and farms and even equipment. Every user can access the task list on his/her dashboard with a kanban view.
- Batch Firmware Updates
 Change or apply Firmware Updates in bulk for a group of devices or farms.
- Customer and hosting services support
 Give secure access to hosting customers.
Ebit/Ebang Miners Auto freq (pll) management :
- Automatic tuning for Ebit Miners


Enterprise ( For bigger Farms ):
- Dedicated Server
- Local offline installations
 A dedicated server that is personalized and optimized for your organization.
- Customization support
 Support for adding features or changes.

Does your software support bosminer? There is growing interest from the Braiins OS community for centralized monitoring tools; and apparently, the usual providers have been slow to adapt their software to add the different api fields added/changed from cgminer.

If your dev team needs help to implement bosminer support, tell them to visit the Braiins OS Telegram support group. The api is not formally documented, but you could inspect bosminer source code or ask if you have specific questions.

If you have an S9 with Braiins OS or Braiins OS+ installed, you might find interesting to inspect this url from your web browser:
http://your.min.er.ip/cgi-bin/luci/admin/miner/api_status

The most important thing of that json dump, is that it shows the api commands you can send to port 4028, so for example doing this on linux:

echo -n '{"command":"tunerstatus"}' | ncat 10.0.0.2 4028 | jq | grep '"PowerLimit"\|"HashchainIndex"\|"Status"'

Gives you:

          "HashchainIndex": 6,

          "HashchainIndex": 7,
          "Status": "Tuner not running",
